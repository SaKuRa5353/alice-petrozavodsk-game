# Alice Petrozavodsk Game

Учебный проект Викиверситета: голосовой навык для Алисы
"Угадай достопримечательность Петрозаводска".

## О проекте

Приложение запускает викторину на 5 раундов.
В каждом раунде пользователь получает описание и подсказку, затем пытается угадать
достопримечательность Петрозаводска.

Проект разделен на независимый игровой движок и webhook-обработчик для Яндекс Диалогов,
поэтому логику легко тестировать локально через CLI.

## Ключевые возможности

- 5 случайных вопросов в одной игре.
- Проверка ответов по основному названию и алиасам.
- Команды управления:
	- `помощь` - показать правила.
	- `сдаюсь` - показать правильный ответ и перейти к следующему вопросу.
	- `заново` - начать новую игру.
- Итоговый счет после завершения всех раундов.

## Технологии

- Python 3.11+
- Стандартная библиотека Python (без обязательных внешних зависимостей)
- Yandex Dialogs webhook API

## Архитектура

- `game_engine.py` - ядро игры: состояние, нормализация текста, проверка ответов, раунды.
- `landmarks.py` - датасет достопримечательностей и источники.
- `alice_handler.py` - webhook-обработчик для Яндекс Диалогов.
- `demo_cli.py` - локальный консольный запуск для быстрой проверки.

## Быстрый старт

### 1) Клонирование и вход в проект

```bash
git clone https://github.com/SaKuRa5353/alice-petrozavodsk-game.git
cd alice-petrozavodsk-game
```

### 2) Локальный запуск через CLI

```bash
python -m venv .venv
source .venv/bin/activate
python demo_cli.py
```

Команда выхода из CLI: `exit`.

## Webhook для Яндекс Диалогов

Точка входа: `handler(event, context)` в `alice_handler.py`.

Минимальный пример входного события:

```json
{
	"request": {
		"original_utterance": "старт"
	},
	"session": {
		"new": true,
		"user_id": "user-123"
	}
}
```

Шаги подключения:

1. Создайте навык в Яндекс Диалогах.
2. Выберите формат с webhook URL.
3. Разверните `alice_handler.py` (например, в Yandex Cloud Functions).
4. Укажите URL функции в настройках навыка.
5. Проверьте сценарии `помощь`, `сдаюсь`, `заново`.
