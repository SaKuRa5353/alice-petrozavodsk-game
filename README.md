## Описание проекта
Навык предлагает пользователю игру из 5 раундов.
В каждом раунде Алиса дает описание достопримечательности Петрозаводска, а пользователь пытается угадать ее название.

Поддерживаются команды:
- `помощь` — правила и список команд
- `сдаюсь` — показать правильный ответ и перейти к следующему вопросу
- `заново` — начать новую игру с нуля

После 5 вопросов показывается итоговый счет.

## Стек
- Python 3.11+
- Логика игры без привязки к платформе в `game_engine.py`
- Обработчик для Яндекс Диалогов (Webhook / Cloud Function) в `alice_handler.py`

## Структура репозитория
- `landmarks.py` — данные по достопримечательностям
- `game_engine.py` — механика викторины
- `alice_handler.py` — обработка запросов от Яндекс Диалогов
- `demo_cli.py` — локальный запуск в консоли
- `wikiversity_quiz.wiki` — готовая викторина для страницы Викиверситета
- `teacher_email_template.md` — шаблон письма преподавателю

## Быстрый старт (локально)
1. Установите Python 3.11+.
2. Запустите консольную проверку:

```bash
python demo_cli.py
```

## Подключение к Яндекс Диалогам
1. Создайте навык в Яндекс Диалогах.
2. Выберите тип: навык с webhook URL.
3. Разверните `alice_handler.py` в облаке (например, Yandex Cloud Functions).
4. Вставьте URL функции в настройках навыка.
5. Протестируйте команды: `помощь`, `сдаюсь`, `заново`.
